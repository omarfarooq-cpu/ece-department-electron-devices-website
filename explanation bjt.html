1. Introduction to BJT

A Bipolar Junction Transistor (BJT) is a three-layer semiconductor device used for amplification and switching applications.

It consists of three regions:

  • Emitter (E)
  • Base (B)
  • Collector (C)

Transistor Construction

The BJT is constructed by sandwiching a thin layer of one type of semiconductor material (the base) between two layers of the opposite type (the emitter and collector). The emitter is heavily doped to inject a large number of charge carriers, while the collector is lightly doped to collect these carriers. The base is very thin and lightly doped, allowing most of the carriers from the emitter to pass through to the collector.

2. Types of BJT

NPN Transistor

In NPN transistor, current is mainly due to electrons.

PNP Transistor

In PNP transistor, current is mainly due to holes.

3. Working Principle

In a BJT, the emitter injects charge carriers into the base. The base is very thin, so most carriers move into the collector.

  • Emitter-base junction is forward biased
  • Collector-base junction is reverse biased

4. Current Components

In a transistor:

  • Emitter current (IE)
  • Base current (IB)
  • Collector current (IC)

Relationship:

IE = IB + IC

5. Transistor Configurations

Common Base (CB)

  • Low input resistance
  • High output resistance
  • Used in high-frequency applications

Common Emitter (CE)

  • Most widely used
  • High gain
  • Used in amplifiers

Common Collector (CC)

  • Used as buffer
  • High input resistance

6. Input and Output Characteristics

Characteristics show the relationship between voltage and current in different configurations.

  • Input characteristic: IB vs VBE
  • Output characteristic: IC vs VCE

7. Hybrid Model

The hybrid model is used to analyze transistor circuits using parameters:

  • hie (input impedance)
  • hfe (current gain)
  • hoe (output admittance)
  • hre (reverse voltage gain)

8. Ebers-Moll Model

The Ebers-Moll model represents the transistor using two diodes and current sources. It explains transistor operation in all regions.

9. Biasing Techniques

Biasing is used to set the transistor operating point.

  • Fixed bias
  • Collector-to-base bias
  • Voltage divider bias

10. Q-Point and Stability

Q-point (Quiescent point) is the operating point of the transistor. It should be stable for proper amplifier operation.

11. Load Line Analysis

Load line represents all possible operating points of a transistor.

  • DC load line
  • AC load line

12. Early Effect

Early effect is the variation of collector current due to change in collector-base voltage.

13. Thermal Runaway

Increase in temperature increases collector current, which further increases temperature. This may damage the transistor.

14. Applications

Amplifier

BJT is used to amplify weak signals.

Switch

BJT works as ON/OFF switch in digital circuits.

Learning Outcome

  • Understand BJT operation
  • Analyze transistor characteristics
  • Design amplifier circuits
  • Use BJT as a switch
